**Student:** Nguyen Minh Tuan | **Date:** October 14, 2025

## Overview
Lab 2 implements modern CSS layout techniques using Flexbox and Bootstrap framework across three exercises.

## Exercises

### Exercise 1: Bootstrap Survey Form
- ✅ Bootstrap 5 integration with CDN
- ✅ Responsive form with text inputs, radio buttons, checkboxes, dropdown
- ✅ University logo and custom styling
- ✅ Mobile-first responsive design

**Key Components:** `form-control`, `form-check`, `form-select`, Bootstrap grid system

### Exercise 2: 3-Column Flexbox Layout
- ✅ Semantic HTML5 structure (`header`, `main`, `aside`, `footer`)
- ✅ Fixed-width (980px) centered layout
- ✅ Flexbox implementation with responsive behavior
- ✅ Clean content organization

**CSS Implementation:**
```css
.content {
  display: flex;
  gap: 20px;
  align-items: stretch;
}
.main {
  flex: 1 1 auto;
}
```

### Exercise 3: Town Hall Website
- ✅ Professional layout with logo integration
- ✅ Two-column content area (main + sidebar)
- ✅ Colored speaker links with CSS classes
- ✅ Responsive design with mobile stacking

**Features:** Dynamic speaker listings, lecture information, contact details

## Technical Skills
- **Flexbox**: `display: flex`, `justify-content`, `align-items`, `gap`
- **Bootstrap**: CDN integration, grid system, form components
- **Responsive Design**: Media queries, mobile-first approach
- **Semantic HTML**: Proper document structure and accessibility
